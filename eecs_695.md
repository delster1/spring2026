# eecs 695 - Software Reverse Engineering

## Syllabus

**class info** 
Prereq's:
- 388 
- c/c++


**course objectives** 
- explain basics of binary software - layout, structure, execution
- explain basics of reverse engineering & binary analysis - disassembly, instrumentation
- learn relevant cli tools & frameworks for reverse engineering
- practice reverse engineeringn using CTF examples & projects

**course structure** 
- basic concepts explained + demonstrated in class using examples
    - understand in class + practice later
- homeworks assigned that evaluate understanding + skills

**course topics** 
- What is software reverse engineering?
- Software build process
- Software run process
- Basic Linux command line tools
- Assembly language basics
- Binary layout (ELF)
- Static reverse engineering tools
- Dynamic reverse engineering tools
- HLL constructs in assembly/binary
- Binary disassembly
- Basic methods for binary analysis
- Static and dynamic code injection and binary instrumentation
- Other advanced topics, like binary decompilation, packing, obfuscation, etc.
